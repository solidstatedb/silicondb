Detailed Description

template<typename T, typename TOwner, T(TOwner::*)() const creator>
class CachedItem< T, TOwner, creator >

Wrapper for data that needs to be cached. The cached data can be accessed via the get() method from multiple threads. The first thread that calls get() will trigger creation of the data item via the creator method, blocking other threads until the item is available in the cache.

Template Parameters
Tthe type of the data item in the cache.
TOwnerthe class containing the cached item.
creatorthe method of TOwner to call in order to create the data item to be cached.
